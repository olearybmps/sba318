# Per Scholas: SBA318 Express Server Application
## Little League Team Data
HTML, CSS, and Javascript project for web development

## Usage
The website allows users to :
- View information
- Utilize API to get team data
    - Roster
    - Schedule
    - Hitting Stats
    - Pitching Stats
- Add a player
- Edit a player
- Delete a player

##Technologies Used
- HTML5
- CSS
- Javascript
- Node.js
- Express
- React
